1)список консольных команд(<> не учитывать):

print <значение> - возвращает значение
calc <выражение> - считает значение выражения

2)список команд(<> не учитывать):

prompt <имя переменной> <тип данных> - принимает значение от пользователя и записывает его в указанную переменную в указанном типе данных

var set <имя переменной>:<значение> <тип данных> - задает указанной переменной значение в нужном типе данных

var show <имя переменной> - возвращает содержимое переменной

random <имя переменной> /<минимальный порог> :<максимальный порог> - создает случайное число в указанном диапазоне и сохраняет в указанную переменную

random <имя переменной> input - принимает два значения от пользователя и в их диапазоне генерирует случайное число которое записывает в указанную переменную
